CDD INGÉNIEUR D'ÉTUDES Institut national de recherche pour l'agriculture, l'alimentation et l'environnement (INRAE) Unité Technologies et systèmes d'information pour les agrosystèmes (TSCF) Centre Clermont Auvergne-Rhône-Alpes Cette mission avait pour objectif l'étude de deux approches de couplage d'un bras robot manipulateur avec une base mobile à roue.
Développements réalisés
- Mise en œuvre de l’architecture LambdAgrIoT - Configuration minimale d’un server Kafka - Interfaçage de topics ROS avec Kafka - Développement d'une interface web de supervision dans le cadre du projet SuperRob: -> Affichage d'information GPS à l'aide d'OpenStreetMap -> Affichage d'informations du robot (vitesse, algorithme en cours de fonctionnement, -> Mise en oeuvre d'achitecture web d’échange de données entre le robot et l’interface - Application à différents types de robot : -> Robot Campero Scénarii réalisés Raliemment d'un point
Suivi d'une trajectoire sinusoidale
Suivi d'une trajectoire discontinue - Visualisation distante en conditions dégradées (Robots dans l’allier, Ordinateur de supervision en Champagne au milieu des vignes)
